Carte d'état-major [Quarto]

Posted on Jan 1, 1

À propos

Exemple de rendu d’un fichier Quarto .qmd, avec un code R exécutable affichant la carte d’état major (ci-dessous), et l’inclusion d’une vue cartographique plus grande dans le panneau de droite à partir du frontmatter (cf. l’article Hugo)

Code 
library(dplyr)
library(leaflet)
leaflet(width = "100%") %>% 
  setView(lat = 47, lng = 2.333, zoom = 6) %>% 
  addWMSTiles(
    "https://data.geopf.fr/wms-r/wms",
    layers = "GEOGRAPHICALGRIDSYSTEMS.ETATMAJOR40",
    options = WMSTileOptions(
      format = "image/png",
      transparent = TRUE,
      version = "1.3.0"
    ),
    attribution = "IGN",
    group = "Carte d'état-major"
  )
